KEY INFORMATION SUMMARY
=======================

WHAT IS BEING BUILT:
  WhatsApp integration for n8n workflow that receives messages and voice notes,
  extracts intent using Claude AI, routes to Notion databases, and sends
  confirmation messages back to users.

INTENT TYPES SUPPORTED:
  job_setup - Create new job (e.g., "new job for John Smith")
  cost_log - Log material costs (e.g., "cost 45 euros")
  labour_log - Log labour hours (e.g., "worked 2 hours")
  invoice_trigger - Generate invoice (e.g., "create invoice")
  payment_received - Record payment (e.g., "payment 500 received")
  unknown - Unrecognized input (e.g., "hello how are you")

TIMELINE:
  Preparation: 15-30 minutes
  Implementation: 20-30 minutes
  Testing: 1-2 hours
  Deployment: 10 minutes
  Total: 3-4 hours

NODES TO CREATE:
  WhatsApp Webhook (trigger)
  Send WhatsApp Response (HTTP request)
  Update: Extract Intent with Claude (modify prompt)

CREDENTIALS NEEDED:
  Twilio Account SID
  Twilio Auth Token
  Claude API key (already configured)

================================================================================
QUICK START: 5 STEPS, 30 MINUTES
=================================

1. Create WhatsApp Webhook node
   - Type: Webhook
   - Path: whatsapp
   - Method: POST
   - Save webhook URL for Twilio

2. Update Intent Extraction node
   - Add WhatsApp context to Claude prompt
   - Update input mapping to handle both sources

3. Create Response Handler node
   - Type: HTTP Request
   - URL: Twilio Messages API endpoint
   - Auth: Basic Auth with Twilio credentials

4. Connect all nodes
   - WhatsApp Webhook → Extract Intent
   - Route by Intent → Send WhatsApp Response
   - Verify no errors

5. Register webhook in Twilio
   - Go to Twilio Console
   - Paste webhook URL in WhatsApp Sandbox
   - Verify sandbox phone number

See WHATSAPP_QUICK_START.md for detailed steps.

TESTING
=======

6 Unit Test Scenarios:
  1. New job setup: "new job for John Smith"
  2. Cost log: "material cost 45 euros 50"
  3. Labour log: "John worked 2 and a half hours"
  4. Invoice trigger: "please create invoice"
  5. Payment received: "payment of 500 euros received"
  6. Unknown intent: "hello how are you"

Expected results in each scenario:
  - Intent correctly extracted
  - Appropriate Notion entry created
  - WhatsApp response sent
  - Message received on test phone

See WHATSAPP_TEST_DATA.md for webhook payloads and expected outputs.

TROUBLESHOOTING QUICK REFERENCE
================================

Problem: Webhook not receiving messages
  Check: Is URL registered in Twilio?
  Check: Is workflow activated?
  See: WHATSAPP_NODE_REFERENCE.md

Problem: Intent classification incorrect
  Check: Is Claude prompt clear?
  Fix: Add more examples to prompt
  See: WHATSAPP_NODE_REFERENCE.md

Problem: Response not sending
  Check: Are Twilio credentials correct?
  Check: Does Account SID match?
  See: WHATSAPP_NODE_REFERENCE.md

Problem: Notion entry not created
  Check: Did handler execute?
  Check: Is Notion connection valid?
  See: WHATSAPP_TEST_DATA.md

More troubleshooting in each document.
